Python Trading Bot

The goal of this project was to construct a python bot that could day trade the SPDR S&P 500 ETF Trust (SPY) in a profitable manner. I connected to the online brokerage Alpaca's API and utilized their paper trading features in Python. I implemented an algorithm which utilizes the high-risk Martingdale system to buy and sell shares of SPY during trading hours to hopefully increase the overall portfolio value daily
