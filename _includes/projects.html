Project Title

The goal of this project was to construct a static website to host my perosnal projects and resume. I began by creating a Github repository and hosting a free page with Github pages using HTML and CSS. I realized that I wanted to automate some of the tedious steps in static site generation so I implemented Jekyll along with an open source theme. After uploading the theme I added some customization using JavaScript and Devicon icons which you can see throughout the website along with some minor aesthetic fixes. The next step in this project is to buy a domain name with hosting.
